# LeKeFu WordPress客服插件

LeKeFu是一款简单易用的WordPress客服插件，支持在网站上展示自定义的客服二维码图片，帮助网站访客快速联系客服。

## 功能特点

- 支持启用/禁用插件
- 支持上传自定义客服二维码图片
- 灵活的显示位置选择（左下角、底部中间、页面中间、右下角）
- 可自定义描述文字
- 可设置显示延迟时间
- 支持排除特定分类和文章
- 响应式设计，完美适配移动端
- 优雅的关闭按钮
- 平滑的动画效果

## 安装说明

1. 下载插件压缩包
2. 登录WordPress后台，进入"插件">"安装插件"页面
3. 点击"上传插件"，选择下载的压缩包
4. 安装完成后，点击"启用插件"

或者：

1. 将插件文件夹 `lekefu` 上传到 `/wp-content/plugins/` 目录
2. 在WordPress后台插件页面启用插件

## 使用说明

### 基本设置

1. 在WordPress后台，进入"设置">"lekefu设置"
2. 勾选"启用插件"选项
3. 点击"选择图片"按钮，上传或选择客服二维码图片
4. 选择显示位置：
   - 左下角
   - 底部中间
   - 页面中间
   - 右下角
5. 可选填写描述文字，将显示在二维码图片下方
6. 设置延迟显示时间（单位：秒，0表示立即显示）
7. 点击"保存更改"按钮

### 高级设置

#### 排除特定页面

您可以设置在特定分类或文章页面不显示客服二维码：

1. 排除分类：
   - 在"排除分类"文本框中输入分类ID
   - 多个分类ID用英文逗号分隔
   - 例如：1,2,3

2. 排除文章：
   - 在"排除文章"文本框中输入文章ID
   - 多个文章ID用英文逗号分隔
   - 例如：100,101,102

## 常见问题

**Q: 如何获取分类ID或文章ID？**
A: 在WordPress后台编辑分类或文章时，查看浏览器地址栏的URL，其中的数字就是ID。
例如：`post.php?post=123&action=edit` 中的 "123" 就是文章ID。

**Q: 为什么客服图片没有显示？**
A: 请检查：
1. 插件是否已启用
2. 是否已上传客服二维码图片
3. 当前页面是否在排除列表中
4. 是否设置了延迟显示时间

**Q: 如何调整客服图片大小？**
A: 插件会自动调整图片大小，默认最大宽度为200px，在移动端会自动缩小至150px。
